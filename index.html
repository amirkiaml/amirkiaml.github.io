Book Tensorflow Recommender System

For my Streamlit dashboard book recommender project, I developed a sophisticated model leveraging TensorFlow and TensorFlow Recommenders (TFRS) to provide personalized book recommendations. The project is structured to address retrieval and ranking tasks, ensuring a two-step suggestion refinement process. Initially, the retrieval phase focuses on narrowing the vast space of possible books to a shortlist that aligns with the user's interests. This is achieved through user and book embeddings, capturing the nuanced relationships between users, authors, and book titles based on historical interactions and preferences.

A key feature of this project is its implementation within a Streamlit dashboard, offering an interactive and user-friendly interface. Users can input their details, such as user ID, preferred author names, and age, to receive tailored book recommendations. The backend model dynamically adapts to user inputs, showcasing the flexibility and responsiveness of the system.

Safety Recommender System for Oil and Gas Industry

Project Overview:

As part of a dynamic team of four, I contributed to developing a Safety Recommender System for a leading oil and gas company. This project aimed to enhance workplace safety by providing actionable safety recommendations, leveraging advanced data extraction and analysis techniques.

Challenges Addressed:

  • Data Extraction from Complex Documents: We tackled the challenge of extracting text from approximately 10 PDF documents, each containing around 150 pages of critical safety reports.
  • Integration of Diverse Technologies: Our approach integrated multiple technologies to ensure comprehensive data extraction and analysis.
  • Development of a User-Friendly Recommender Dashboard: We aimed to make safety recommendations accessible and actionable for the company's personnel.

Solution Pipeline:

  1. Data Extraction Pipeline:
    • Initial Extraction: Utilized Azure Form Recognizer and PymuPDF for robust text extraction from PDF files. This dual approach ensured high accuracy and completeness in text data retrieval.
    • Data Combination and Structuring: The extracted text data from both sources were compared, combined, and structured into a JSON file, creating a standardized format for further processing.
    • Image Extraction: Employed PymuPDF to extract images from the reports, organizing them in a folder structure based on the Year and page numbers, facilitating easy access and correlation with textual data.
  2. Dataset Creation: Developed a comprehensive dataset encapsulating incident reports alongside corresponding image paths. This dataset served as the foundation for our analysis and recommendation generation.
  3. Recommender Dashboard Development: Designed and implemented a user-friendly dashboard to deliver safety recommendations. This tool allows company personnel to easily access and implement safety measures based on historical data and incident analysis.

Technologies Used:

  • Microsoft Azure
  • Azure Form Recognizer
  • PymuPDF
  • JSON for data structuring
  • Dashboard development technologies (Streamlit)

Skills Demonstrated:

  • Team collaboration and project management
  • Advanced data extraction and processing
  • Analytical thinking in developing practical safety solutions
  • Technical proficiency in utilizing Microsoft Azure, Azure Form Recognizer, PymuPDF, and data structuring techniques
